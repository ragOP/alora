# Halsten Stride - React Application

A modern React application converted from HTML/CSS with clean code structure and meaningful CSS class names.

## Project Structure

```
react-app/
├── src/
│   ├── components/          # Reusable components
│   │   ├── Button.jsx
│   │   ├── Container.jsx
│   │   ├── ImagePlaceholder.jsx
│   │   └── VideoPlaceholder.jsx
│   ├── sections/            # Page sections
│   │   ├── HeroSection.jsx
│   │   ├── ProblemSection.jsx
│   │   ├── SolutionSection.jsx
│   │   ├── TestimonialSection.jsx
│   │   └── CtaSection.jsx
│   ├── styles/              # Global styles
│   │   ├── index.css
│   │   └── App.css
│   ├── App.jsx              # Main app component
│   └── main.jsx             # Entry point
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## Features

- ✅ Clean React components
- ✅ Meaningful CSS class names (no generated names)
- ✅ Modern CSS with CSS variables
- ✅ Responsive design
- ✅ Component-based architecture
- ✅ Placeholders for images and videos

## Installation

```bash
cd react-app
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Technologies

- React 18
- Vite
- Modern CSS with CSS Variables
- Responsive Design
